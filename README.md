# flutter template

一个完整得flutter项目模板

* `api` http(接口请求)封装
* `routers` fluro(路由)封装
* `utils` custom utils (常用工具)封装
* `store` Provider （类vuex数据）封装
* `localstorage` shared_preferences （本地数据存储）封装

## enviroment

flutter version: 1.12.13+hotfix.8

## Getting Started

```dash
# 运行
flutter run

# 打包，注意需要提前准备好签名
# flutter build apk
```
